# 📖 설치 및 실행 가이드

## 1. 사전 요구사항

- Node.js 18.17 이상
- npm 또는 yarn 패키지 매니저

## 2. 설치

```bash
# 프로젝트 폴더로 이동
cd 20260308_사진명함철

# 의존성 설치
npm install
```

## 3. 실행

### 개발 모드 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

### 프로덕션 빌드

```bash
npm run build
npm start
```

## 4. 주요 기능 사용법

### 명함 추가하기

1. 홈 화면 우하단의 **+** 버튼 클릭
2. 명함 이미지 업로드 (드래그 앤 드롭 또는 클릭)
3. **OCR 자동 인식 시작** 버튼 클릭 (또는 건너뛰기)
4. OCR 결과 확인 및 수정
5. **명함 저장** 버튼 클릭

### 명함 보기

1. 홈 화면에서 명함 카드 클릭
2. 상세 정보 모달에서 확인
3. 전화번호/이메일 클릭 시 자동 연결

### 회사별 필터링

1. 홈 화면 상단의 회사 필터 드롭다운 사용
2. 특정 회사 선택 시 해당 회사 명함만 표시

### 페이지 넘기기

- 하단의 **이전/다음** 버튼 사용
- 현재 페이지 / 전체 페이지 표시

### 데이터 백업

1. 홈 화면 상단의 다운로드 아이콘 클릭 (내보내기)
2. JSON 파일로 저장

### 데이터 복원

1. 홈 화면 상단의 업로드 아이콘 클릭 (가져오기)
2. 백업한 JSON 파일 선택

## 5. 문제 해결

### OCR이 작동하지 않을 때

- 인터넷 연결 확인 (Tesseract.js는 언어 데이터를 다운로드합니다)
- 브라우저 콘솔에서 에러 확인
- 이미지가 너무 크면 시간이 오래 걸릴 수 있습니다
- **OCR 건너뛰고 직접 입력** 사용

### 명함이 저장되지 않을 때

- 브라우저의 IndexedDB 지원 확인
- 시크릿 모드에서는 저장이 제한될 수 있습니다
- 브라우저 저장 공간 확인

### 이미지가 표시되지 않을 때

- 이미지 파일 형식 확인 (JPG, PNG 권장)
- 파일 크기가 너무 크지 않은지 확인
